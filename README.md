## Index of Techniques Used

|--------------|----------------------------|

This project is is screencast of me scraping data from an open bible site. I use this site to get different versions of the bible so I might compare their lengths. Using interactive graphics, I am to show these differences and teach others on how to use plotly and R shiny.
